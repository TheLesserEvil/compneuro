        • Something to consider...

          When handing in the exercises, please keep in mind the following:


          • provide the m-files
          • make sure the m-files give meaningful output (plots, printed variables)
          • plots should have labels, legends, titles
          • provide evidence for your statements: if we can not reconstruct what you did, we can not give point for individual steps, reducing your score
          • give your files names that uniquely relate to the exercises (so 'week1_ex1b_newton', not 'ode1' or 'output2')
          • if you have many files, consider uploading a zip
          • make sure the code runs; from this week on, code that doesn't run, no matter what the cause is, is awarded 0 points
          • hand in on time (11pm sharp)
